2a. Numerical Integration (30 pts)¶

For a domain discretized into $N$ equally spaced panels: $$ \Delta x=\frac{b-a}{N}. $$ The approximation to the integral is $$\begin{aligned} \int_a^b f(x) d x & \approx \frac{\Delta x}{2} \sum_{k=1}^N\left(f\left(x_{k-1}\right)+f\left(x_k\right)\right) \\ & =\frac{\Delta x}{2}\left(f\left(x_0\right)+2 f\left(x_1\right)+2 f\left(x_2\right)+2 f\left(x_3\right)+\cdots+2 f\left(x_{N-1}\right)+f\left(x_N\right)\right) \\ & =\Delta x\left(\sum_{k=1}^{N-1} f\left(x_k\right)+\frac{f\left(x_N\right)+f\left(x_0\right)}{2}\right) . \end{aligned}$$ This is the trapezoid rule scheme.

# values of n to be used for calculations using trapezoid rule
n_list = [10, 20, 40, 80, 160, 320, 640, 1280]

# constant values
a, b = -4, 6

# list to hold the integral estimations for each value of n
results = []

for n in n_list:
    # the change in x from x_k to x_k+1
    # the greater the value of n, the smaller the subdivisions
    #  between a and b
    delta_x = (b - a) / n

    # values of x_N and x_0 to use in the calculation
    xN = b 
    x0 = a
    
    # the summation in the formula starts with x_1 and stops 
    #   at x_N-1
    # x_1 = x_0 + delta_x
    x_curr = x0 + delta_x

    # carry out the summation
    total = 0 
    for i in range(1, n):
        total += fx(x_curr)
        x_curr += delta_x 
    
    total += (fx(xN) + fx(x0)) / 2

    # multiply the current total by delta_x
    total = total * delta_x

    results.append(total)

def check_if_multiples(row_1: np.array, row_2: np.array) -> bool:
    "returns True if row_2 is a multiple of row_1, False if not"
    tolerance = 0.001

    # to avoid division by 0 error, do a separate check for 0s in each row
    r1_0s = r2_0s = np.array([]).astype(int)

    # if 0 is in only one row, then the rows cannot be multiples
    if (0 in row_1) != (0 in row_2):
        return False
    
    # if 0 is in both rows, then check the indices of the 0s in each row
    if (0 in row_1) and (0 in row_2):
        r1_0s = np.where(row_1 == 0)[0]
        r2_0s = np.where(row_2 == 0)[0]
        
        # if the '0' indices for both rows don't match, then they are not multiples
        if r1_0s.shape != r2_0s.shape or not all(r1_0s == r2_0s):
            return False 
    
    # if any 0s were found and their indices match, delete them before dividing the rows
    # and then continue normally
    div = np.delete(row_2, r2_0s) / np.delete(row_1, r1_0s)
    if max(div) - min(div) < tolerance:
        return True 
    return False

def LUdecomposition(A: np.array) -> [np.array, np.array]:
    # takes (n x n) matrix A as an input and returns a list
    # containing the resulting L and U matrices from 
    # the LU decomposition process

    # check if the input array has exactly two dimensions
    if len(A.shape) != 2:
        raise InvalidShapeException("Input A is not 2-dimensional")

    # check if the input array is square shape
    rows, cols = A.shape
    if rows != cols:
        raise InvalidShapeException("Input A is not (n x n)")
    
    # # check if there are any 0s present in the main diagonal
    # if 0 in A.diagonal():
    #     raise NeedsPivotException("Pure LU Decomposition does not exist " + 
    #                               "for input A. Pivoting is needed.")
    
    # check if any rows are multiples of each other 
    # for i in range(rows):
    #     for j in range(i + 1, rows):
    #         if check_if_multiples(A[i], A[j]):
    #             raise RowsCancelledOutException(f'{A[i]} and {A[j]} are multiples. They will reduce to 0s.')

    # matrix U starts off identical to the input A
    # matrix L starts off as the identity matrix
    U = A.copy().astype(float)
    L = np.identity(rows)

    for i in range(rows):
        # get the leading term for that row (the term on the main diagonal)
        lead = U[i, i]
        if lead == 0:
            raise NeedsPivotException('Encountered a 0 as pivot element. ' + 
                                      'This matrix does not have a pure ' + 
                                      'LU Decomposition.')
        
        # want to reduce all the terms below the leading term to 0
        for j in range(i + 1, rows):
            # calculate the multiplier for the current term that will
            # allow it to sum to 0 with each term below it 
            multiplier = float(-U[j, i] / lead) 
 
            # multiply the current row by the scalar multiplier and then
            # add this row to the remaining iterated rows
            multiplied_row = U[i] * multiplier 
            U[j] = U[j] + multiplied_row 

            # the element in array L at the current row and column will be
            # the negated version of the multiplier
            L[j, i] = -multiplier
    
    return [L, U]

def areMatricesEqual(A: np.array, B: np.array, tolerance: float = 0.0001):
    # checks if matrices A and B are equal to each other, allowing for slight error
    # if |A[i,j] - B[i,j]| <= tolerance for every i,j, return True; else False
    diff = np.abs(A - B)
    if np.max(diff) <= tolerance:
        return True
    return False

pivoting() function for PLU Decomposition¶

In [ ]:
def pivoting(A: np.array) -> np.array:
    # takes an input matrix A and returns the permutation/pivoting
    # matrix P such that PA has a pure LU decomposition

    rows, cols = A.shape
    if rows != cols:
        raise InvalidShapeException("Input A is not (n x n)")
    
    temp = A.copy().astype(float)
    P = np.identity(rows)

    for i in range(cols):
        lead = temp[i, i]
        # find the row index of the largest element in the current column
        swap_index = max(range(i, cols), key=lambda r: temp[r, i])

        # swap the current row with the row containing the max element
        temp[[i, swap_index]] = temp[[swap_index, i]]
        P[[i, swap_index]] = P[[swap_index, i]]

        # do gaussian elimination on the current version of the matrix
        # and repeat the process with the new pivots
        for j in range(i + 1, rows):
            multiplier = float(-temp[j, i] / lead)
            mult_row = temp[i] * multiplier 
            temp[j] = temp[j] + mult_row 


    return P

6. Markov Chain (30 pts)¶

BACKGROUND: Consider a Markov chain transition matrix $P$ on a set of $n$ states, where $P_{i j}$ corresponds to the probability $\left(0 \leq P_{i j} \leq 1\right)$ to go from state $i$ to the state $j$, and each row is normalized so that $$ \sum_{j=1}^n P_{i j}=1 . $$

Let $p$ be a size- $n$ vector composed of the probability distribution over the $n$ states, where $$ \sum_{j=1}^n p_j=1 . $$

The transition rule corresponding to the transition matrix $P$ is $\hat{p}=P^T p$, where $\hat{p}$ corresponds to a new probability distribution and $$ \sum_{j=1}^n \hat{p}_j=1 . $$ TASK: Write a program that works with 5 states only use the numpy library. In particular, 0. Fix the random seed to be the last digit of your id: numpy.random.seed().

  1. Construct a random $5 \times 5$ matrix $P$, and normalize each row so that $$ \sum_{j=1}^5 P_{i j}=1 . $$

  2. Construct a random size-5 vector $p$ and normalize it so that $$ \sum_{j=1}^5 p_j=1 . $$ Apply the transition rule 50 times to obtain $p_{50}$.

  3. Compute the eigenvector $v$ of $P^T$ corresponding to the eigenvalue 1 (and numerically, the eigenvalue closest to $1)$, and then scale the eigenvector so that $$ \sum_{j=1}^5 v_j=1 . $$ This scaled eigenvector is known as the stationary distribution.

  4. Compute the component wise difference between $p_{50}$ and the stationary distribution. Do they match with each other within $1 \mathrm{e}-5$ ?

HINTS: You may find the following functions useful in your implementation:

  • np.random.rand
  • np.dot
  • np.linalg.eig

# transition rule: p* = (P^T)p
def transition(P: np.array, p: np.array) -> np.array:
    # takes a Markov chain transition matrix P and probability distribution 
    # vector p as inputs and returns a new probability distribution p_new
    p_new = np.matmul(P.T, p)
    return p_new
